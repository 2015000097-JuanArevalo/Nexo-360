# NEXO 360 - Entregables

Este paquete contiene los entregables principales para los primeros dos hitos del proyecto NEXO 360.

Está escrito de forma simple porque el proyecto es para nivel escolar, no como documentación formal de ingeniería de software.

## Archivos incluidos

1. `01_Alcance_del_Proyecto.md`  
   Define qué es NEXO 360, el objetivo del proyecto, el problema que resuelve, el público al que va dirigido y qué debe quedar listo para la entrega.

2. `02_Lista_de_Funciones_MVP.md`  
   Separa las funciones principales que sí deben estar en la primera versión de las funciones que pueden quedar para después.

3. `03_Roles_y_Accesos.md`  
   Explica qué puede hacer cada tipo de usuario dentro de la plataforma.

4. `04_Plan_de_Demo_y_Checklist.md`  
   Presenta una guía sencilla para demostrar el proyecto y una lista de verificación antes de entregarlo.

## Uso recomendado

Estos documentos pueden usarse como base para:
- Presentación del proyecto.
- Informe escrito.
- Organización del equipo.
- Revisión de avance antes de la fecha límite.
